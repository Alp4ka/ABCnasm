### Запуск:
Стандартный:
```sh
./runner input.txt output.txt
```


Рандомная генерация (-r):
```sh
./runner -r 34 input_file.txt output_file.txt
```
> **Важно:** в input_file.txt запишется результат рандомной генерации 34 элементов.
